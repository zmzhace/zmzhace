### 🛠️ 正在进行的工作 (Latest Activity)

目前我正全力投入于 **[Open-Cowork](https://github.com/zmzhace/open-cowork)** 的核心攻坚。这不仅仅是一个跨平台协同工作空间，更是我向 **AGI (通用人工智能)** 迈进的实验场。近期的核心研发方向包括：
